# Accident Elves — AccidentElves.com

Las Vegas 24/7 accident help service.

## Setup

```bash
npm install
npm run dev
```

## Deploy to Vercel

1. Push this repo to GitHub
2. Import project at vercel.com
3. Connect AccidentElves.com domain in Vercel settings
4. Point IONOS DNS to Vercel (see instructions below)

## IONOS DNS Settings

In your IONOS domain settings, set these DNS records:

| Type  | Name | Value                |
|-------|------|----------------------|
| A     | @    | 76.76.21.21          |
| CNAME | www  | cname.vercel-dns.com |
